# Plan A vs Plan B — 로티 인수 구조안 v3

> 작성: 2026-03-26 / CSO Leo 지시
> 공통: 함안 설비이전 완료(4/1) → 4개월 샘플 생산(~9월 중순) → 인수 실행
> 비용 선지급: (주)사람들 → 추후 신규법인이 부담

---

## 배경: 왜 두 가지 안인가

### 핵심 문제 — TIPS와 유대표의 관계

TIPS(기술창업지원 프로그램)는 신규법인의 핵심 자금원(~10억, 2027년)이다. 그런데 TIPS에는 엄격한 **창업기업 요건**이 있다:

1. **창업팀이 60% 이상 지분** 보유
2. **모회사(특관자 포함)는 40% 이하**
3. **창업기업확인서** 발급 가능해야 함
4. **창업시점 카운팅**: 대표자가 관여한 법인이 **존속하는 한**, 그 법인의 설립일이 창업시점

**유승덕 대표의 제약:**
- 유대표는 (주)사람들(2021.01.21 설립)의 대표이자 설립자
- 유대표가 사람들을 퇴사하더라도, **(주)사람들 법인이 존속하는 한** 창업시점은 2021.01.21로 카운팅
- 로티처럼 법인 자체가 **소멸**해야 비로소 창업시점이 리셋됨
- 따라서 유대표가 신규법인의 단독 대표가 되면, 신규법인의 창업시점 = 사람들 설립일 → **TIPS 요건 불충족 가능**

이 제약이 A안과 B안의 구조를 근본적으로 다르게 만든다.

### 유대표의 전환 전제

양쪽 안 모두 공통 전제로, 유대표는 **사람들의 지분을 기존 주주들에게 배분**하고 신규회사 쪽으로 이동하는 구조다. 이는 단순 퇴사가 아니라, 사람들에서의 역할을 정리하고 새로운 사업체에 창업 참여하는 개념이다. 다만 법적으로 사람들 법인이 존속하기 때문에, 유대표 명의로는 TIPS 창업기업 확인이 어렵다.

## Plan A: 함안 신규법인 설립 → 로티 인수 (지분교환)

### 핵심 개요

함안에 신규법인을 설립하고, 4개월간 샘플 생산으로 사업 가능성을 확인한 후, **지분교환 방식으로 로티를 인수**하는 구조.

### A안의 TIPS 대표 문제 — 왜 유대표가 대표가 안 되는가

A안에서 신규법인은 독립 법인으로 설립된다. 이때 대표이사를 누구로 하느냐가 TIPS의 핵심이다.

**유대표가 대표가 될 수 없는 이유:**
- 유대표는 (주)사람들의 설립자이자 현 대표
- TIPS 신청 시, 대표자의 창업이력을 국세청 총사업자내역으로 확인
- 사람들이 존속하는 한, 유대표의 창업시점 = 사람들 설립일(2021.01.21)
- 이미 5년차 법인의 대표가 "새로 창업"한 것으로 인정받기 어려움
- **결론: 유대표 단독 대표 시 TIPS 불가**

**해결책 — 대리 대표:**
- 교수님(재료 분야) 또는 유대표의 지인이 대표이사를 맡음
- 유대표는 공동대표 또는 부사장(비대표 임원)으로 참여
- 대리 대표 유의사항: TIPS 창업팀으로 2년 이상 활동 시 **법적 연대책임** 발생
- 대표 후보는 해당 기술 분야 전문성 + TIPS 심사 대응 능력 필요

**유대표의 역할 전환:**
- 유대표는 사람들의 지분을 기존 주주들에게 배분 정리
- 신규법인에 공동대표 또는 부사장으로 합류
- 사람들과 겸직 가능 (겸직확인서 제출)
- 실질적인 사업 총괄은 유대표가 하되, 법적 대표 명의는 다른 사람

### 왜 샘플 확인 후 인수인가

A안의 핵심 특징은 **로티 인수를 9월 중순(샘플/양산기초 확인 후)까지 미루는 것**이다.

- 신규법인이 먼저 설립되고 운영을 시작하지만, 로티 인수는 사업 가능성이 확인된 후에만 실행
- 이는 **인수 실패 리스크를 줄이는** 보수적 접근
- 반면 그 기간 동안 로티의 은행이자(8.05억 × 이율)와 미지급금 이자가 계속 누적되는 단점
- 로티 법인 유지비(최소 세무/법무)도 5개월간 추가 발생

### 지분교환 인수 방식 상세

A안에서 로티 인수는 **지분교환(주식교환)** 방식이다:

1. **로티 주주들이 보유한 로티 주식** → **신규법인 주식으로 교환**
2. 교환 비율은 양사 가치평가에 따라 결정
3. **현금 엑싯 대상** (이보화 21.74% + 야스 18.37%):
   - 이들의 로티 주식을 외부 매수자에게 매각
   - 현금 엑싯 규모: ~3억 이내
   - 매수자 확보가 선행되어야 함
4. **RCPS 기관** (인라이트, 연구개발특구, 하이테크, 에이지):
   - 로티 RCPS → 신규법인 주식으로 교환
   - RCPS 조건(연복리 4%, 상환권 등)의 신규법인 승계 여부 협의 필요
5. **임우빈 대표** (39.85%):
   - 신규법인 주식으로 교환 → 핵심 기술인력+주주로 합류
6. **은행부채 8.05억**:
   - 신규법인이 채무인계 계약으로 승계

```
[Step 5] TIPS 신청 — 10월~
         └── ~10억 (2027년 지원)

[장기] 코스닥 상장
```

### A안 장단점 상세

**장점:**
- 사람들 직접 리스크 최소 — 비용 선지급만, 부채 인수 없음
- 샘플 확인 후 인수 → 사업 실패 시 인수하지 않는 옵션 유지
- 투자를 2단계로 분산 (엔젤 1억 → VC 4~10억) → 밸류에이션 상승 활용
- 신규법인이 독립적으로 시작 → 깨끗한 이력

**단점:**
- **유대표 대표 불가** → 대리 대표 섭외 필수 (2년+ 시 연대책임)
- 대리 대표의 TIPS 심사 대응 능력이 변수
- 인수까지 5개월간 로티 **은행이자+미지급금 이자 계속 누적**
- 로티 법인 유지비 5개월 추가 발생
- 지분교환 시 양사 가치평가 합의가 복잡할 수 있음

## Plan B: (주)사람들 → 로티 인수 → 신규회사가 장비 매입 (채무인계)

### 핵심 개요

(주)사람들이 로티를 주식매입으로 먼저 인수하고, 임우빈+유대표가 신규회사를 설립한 후, 신규회사가 VC 투자금으로 사람들로부터 장비를 채무인계 방식으로 인수하는 구조.

### 왜 B안이 "제일 편한" 구조인가

1. **임우빈이 대표** → TIPS 요건 자동 충족 (임우빈은 로티가 소멸하면 창업시점 리셋)
2. 사람들이 로티를 인수하면 **로티는 소멸** → 임우빈의 창업시점이 리셋됨
3. 유대표도 공동설립자로 참여하되, 대표는 임우빈 → TIPS 문제 없음
4. 구조가 단순: 사람들 인수 → 신규회사 설립 → 장비 이전 → TIPS

### B안의 로티 인수 상세 — 사람들이 왜 부채를 안는가

**인수 대금 구조:**
- 사람들이 로티 주식을 매입할 때, 로티의 부채가 함께 따라옴
- **현금으로 지불하는 부분**: 미지급금 중 긴급한 것 (세금, 이자 등 1순위) + 로티 운영비
- **채무인수 부분**: 나머지 미지급금 + 은행대출 8.05억
- 즉, 사람들이 실제로 큰 현금을 내는 것이 아니라, **로티의 부채를 떠안는** 것이 인수의 핵심
- 이후 신규회사가 장비+은행부채를 채무인계로 가져감으로써, 사람들의 부채 부담이 해소됨

**사람들 투자사 동의가 필요한 이유:**
- 사람들이 로티를 인수하면 일시적으로 부채가 증가 (부채비율 380% → 더 상승)
- 사람들의 기존 투자사(주주)들에게 이 리스크를 설명하고 동의를 받아야 함
- **동의 조건 (예상):**
  - 사람들 지분 일부를 투자사에 추가 배분, **또는**
  - 신규회사 지분을 투자사에 배분
- 이 동의 과정은 유대표가 직접 주주들과 협의하여 결정

**B안이 성립하는 논리:**
- 사람들이 로티를 인수하면서 은행부채를 안지만, **실질적으로 현금이 나가는 것은 아님** (채무인수)
- 신규회사 역시 VC 투자금으로 장비를 인수하면서 **은행부채를 함께 채무인계** → 사람들의 부채 해소
- 전체 트랜잭션이 완료되면: 사람들=로티 흡수 완료, 신규회사=장비+부채+기술 보유, TIPS 신청 가능

### 유대표의 역할 (B안)

- 유대표는 사람들 대표로서 로티 인수를 주도 (6~7월)
- 동시에 임우빈과 함께 신규회사를 **공동대표로 공동설립** (5월 초)
- 사람들의 지분을 주주들에게 배분 정리
- 신규회사에서 유대표+임우빈 = **창업팀**으로 묶임
- **핵심: 유대표가 양쪽을 조율하는 중심 역할, 신규회사에서도 공동대표**

### 임우빈 대표의 TIPS 자격 (B안)

- 사람들이 로티를 인수하면 **로티 법인이 소멸**
- 임우빈은 로티 대표였지만, 로티 소멸 → **창업시점 리셋**
- 신규회사에서 임우빈이 대표이사를 맡으면 → 신규 창업으로 인정
- 창업기업확인서 발급 가능 → **TIPS 신청 자격 충족**
- 이것이 B안의 가장 큰 장점

### 구조도


### B안 핵심 리스크: 사람들 투자사 동의

- 사람들이 로티 인수 시 **채무를 일시적으로 안게 됨** (미지급 4.54억 + 은행 8.05억)
- 사람들의 현재 부채비율이 이미 380% → 추가 부채로 재무건전성 더 악화
- 투자사(주주)들 입장에서는 리스크 증가 → 동의 없이 진행 불가
- **동의 조건 (예상):**
  - 사람들 지분 일부 수령 → 사람들 매출(95.8억)에 대한 업사이드
  - **또는** 신규회사 지분 수령 → 기술사업 업사이드
- 유대표가 주주들과 직접 협의하여 결정
- **완화 요인:** 신규회사가 장비+부채를 빠르게 채무인계하면, 사람들의 부채 부담은 일시적

### 세금 검토 필요 사항 (B안)

B안은 트랜잭션이 여러 단계에 걸쳐 있어 각 단계별 세금 이슈 검토 필수:
1. 사람들의 로티 주식매입 → 취득세, 양도세
2. 장비 매도 (사람들→신규회사) → 부가세, 법인세
3. 채무인계 → 채무면제익 발생 여부
4. 전체 트랜잭션에서 이중과세 방지 구조 설계

### B안 장단점 상세

**장점:**
- **임우빈이 대표** → TIPS 직접 신청 가능 (최대 장점)
- **가장 편한 구조** — 대리 대표 섭외 불필요
- 빠른 진행 (6~7월 인수) → 로티 이자 누적 기간 단축
- 은행부채는 채무인계로 이전 → 실질 현금 부담은 긴급 미지급금만
- 로티 조기 소멸 → 법인 유지비 절감

**단점:**
- **사람들이 일시적으로 부채를 안음** → 재무건전성 악화
- **사람들 투자사(주주) 동의 필요** → 조건 협상이 변수
- 사람들의 부채비율 380% → 추가 부채 시 기업평가등급 하락 우려
- 전체 트랜잭션 세금 구조가 복잡 → 법률/세무 자문 필수
- 채무인계가 지연되면 사람들의 부채 부담 장기화 리스크
