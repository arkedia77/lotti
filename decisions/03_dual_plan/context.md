## TIPS 핵심 요건

| 요건 | 내용 |
|------|------|
| 창업팀 지분 | **60% 이상** |
| 모회사 지분 | **40% 이하** (특관자 포함) |
| 창업기업확인서 | 발급 가능해야 함 |
| 창업시점 | 법인 존속 시 해당 법인 설립일부터 카운팅 |

### 유대표 TIPS 제약
- 유대표가 사람들에서 퇴사해도, 사람들 법인 존속 → **창업시점 = 사람들 설립일(2021.01.21)**
- 로티처럼 법인 소멸해야 다시 카운팅 가능
- **해결책**: 유대표가 아닌 다른 대표 필요 (2년+ 시 창업팀 연대책임)
