# 합병 구조안 — Plan D & Plan E

> 작성: 2026-03-19 / CSO Leo 검토용 초안
> 우선순위: 빠른 실행(c) > 사업 지속(b) > 리스크 최소화(d) > 투자자 exit(a, 상황부)
> 최종 목표: **5,000억+ 벨류 코스닥 상장**

## 전제 조건 정리

| 항목 | 내용 |
|------|------|
| 데드라인 | 4/1 공장 퇴거 (3/21~ 패킹) |

## Plan D: 영업양수도 (Asset Deal)

> **사람들이 로티의 "자산만" 인수, 로티 법인은 정리**

### 구조

```
[로티] ──설비+IP+인력──→ [(주)사람들]
         ↑                    │
    자산매각대금 ←──────────────┘

로티 법인 → 부채 정리 후 청산/휴면
```

### 실행 단계

| 단계 | 내용 | 시기 |
|------|------|------|
| 1 | 사람들 ↔ 로티 영업양수도 계약 체결 | 3월 중 |
| 2 | RCPS 투자자 서면동의 취득 | 3월 중 |
| 3 | 설비 창원 이전 (사람들 공장) | 3/21~4/1 |
| 4 | 은행 담보대출 처리 (사람들이 상환 or 담보 이전) | 4월 |
| 5 | 자산매각대금으로 투자자 부분 exit | 4~5월 |
| 6 | 임대표 CTO로 사람들 합류 | 4월 |
| 7 | 로티 잔여 부채 정리 후 청산 | 5~6월 |

### 자금 흐름 (예시)

| 항목 | 금액 | 비고 |
|------|------|------|
| 은행 대출 상환 | 8억 | 최우선 (설비 담보 해제) |
| 미지급금 처리 | 4억 | 퇴직금 등 |
| 설비 이전비 | 1~1.3억 | 창원 이전 (절감) |
| 투자자 exit 지급 | 협의 대상 | 구주 매각가 × 1:4:2:2 |
| **소계 (인수비용)** | **~15억 + exit금** | |
| 운영비 (12개월) | 월 3~4천만 추정 | 인건비+이자+임차+전력 |
| **총 필요자금** | **25~35억 수준** | exit 규모에 따라 변동 |

### 장점
- **속도**: 주주총회/합병절차 불필요, 계약만으로 실행 가능 → 4/1 대응 가능
- **깔끔함**: 로티의 73.3억 누적결손금, 부채 이력이 사람들에 넘어오지 않음
- **코스닥**: 사람들이 기존 매출(95.8억) + 페로브스카이트 신사업으로 상장 추진
- **비용 범위**: 60~70억 예산 내 충분히 가능

### 단점/리스크
- 투자자 exit 가격 협상이 핵심 난관 (실투입 대비 대폭 할인 불가피)
- 로티 청산 과정에서 대표 가지급 4억 처리 복잡
- 사람들 법인에 "태양광 제조" 사업목적 추가 등기 필요
- 정부과제(TIPS) 신규법인 혜택 활용 불가

## Plan E: 신규 합작법인 (JV) 설립

> **사람들 + 로티 투자자들이 새 법인을 만들고, 설비와 자금을 각각 출자**

### 구조

```
[(주)사람들]───현금 출자──→ [신규법인: (주)로티에너지(가칭)]
                                    ↑
[로티 투자자]──설비 현물출자──→      │  ← 임대표 CTO 합류
                                    │
                           로티 법인 → 청산/휴면
```

### 지분 구조 (예시)

| 출자자 | 출자 형태 | 예상 지분 |
|--------|----------|----------|
| (주)사람들 | 현금 (공장+운영비+대출상환) | 60~70% |
| 로티 투자자 그룹 | 설비 현물출자 (감정가 기준) | 20~25% |
| 임우빈 대표 | 기술출자 + 스톡옵션 | 5~10% |
| 창원대 교수 (추후) | 기술자문/사내이사 | 스톡옵션 |

### 실행 단계

| 단계 | 내용 | 시기 |
|------|------|------|
| 1 | JV 설립 합의 (MOU) | 3월 중 |
| 2 | 신규법인 설립 (자본금 1억~) | 3월 말 |
| 3 | 설비 이전 (사람들 창원 공장으로) | 3/21~4/1 |
| 4 | 로티 → 신규법인 설비 현물출자 (or 양수도) | 4월 |
| 5 | 사람들 현금 출자 (단계적) | 4~5월 |
| 6 | 로티 은행대출 상환 + 법인 정리 | 4~6월 |
| 7 | TIPS 신청 + 연구과제 신청 | 5~6월 |
| 8 | 파일럿 가동 → 샘플 생산 | 하반기 |

### 자금 흐름 (예시)

| 항목 | 금액 | 비고 |
|------|------|------|
| 은행 대출 상환 | 8억 | 로티 부채 정리 |
| 미지급금 | 4억 | |
| 설비 이전비 | 1~1.3억 | |
| 신규법인 운영자금 | 10~15억 | 12개월분 |
| 파일럿 라인 세팅 | 5~10억 | |
| **소계** | **28~38억** | |
| + TIPS 수령 시 | -8억 (2년) | 상쇄 |
| + 연구과제 수령 시 | -3~5억 | 상쇄 |
| **실 부담** | **15~27억** | 정부자금 확보 시 |

### 장점
- **정부자금**: 신규법인 → TIPS 8억 + 연구과제 3~5억 = 최대 13억 확보 가능
- **깔끔한 캡테이블**: 처음부터 목표(코스닥 5천억)에 맞는 지분 구조 설계
- **투자자 유지**: exit 대신 JV 지분으로 전환 → 5천억 상장 시 업사이드 공유
- **임대표 인센티브**: 스톡옵션으로 동기부여 극대화
- **창원대 교수 합류 시**: 정부과제 수주 확률 대폭 상승

### 단점/리스크
- Plan D보다 **설립/구조화에 2~3주 더 소요**
- 설비 현물출자 시 감정평가 필요 (비용+시간)
- 로티 투자자들이 "exit 없이 지분 전환"에 동의해야 함
- 사람들이 대주주인 JV에 대한 기관투자가 신뢰도 이슈

| 기준 | Plan D (영업양수도) | Plan E (JV 신설) |
|------|:---:|:---:|
| **실행 속도** | ★★★★★ | ★★★☆☆ |
| **사업 지속성** | ★★★★☆ | ★★★★★ |
| **리스크 최소화** | ★★★★☆ | ★★★★★ |
| **투자자 exit** | ★★★☆☆ (현금 할인) | ★★☆☆☆ (지분 전환) |
| **정부자금 활용** | ★☆☆☆☆ | ★★★★★ |
| **코스닥 경로** | 사람들 직접 상장 | JV 상장 or 사람들 합병 후 상장 |
| **초기 필요자금** | 25~35억 | 15~27억 (정부자금 시) |
| **4/1 대응** | 가능 | 가능 (설비 이전은 동일) |

## CSO 추천: Plan E (JV) + Plan D 요소 혼합

### 이유
1. **정부자금 13억**은 너무 큼 — 60~70억 예산에서 20% 절감 효과
2. **5천억 코스닥**이 최종 목표 → 깨끗한 신규법인이 IR/상장심사에 유리
3. 투자자들이 동의 의향 있고, Son이 적극적 → 지분 전환 수용 가능성 높음
4. 설비 이전은 어차피 동일하게 진행 가능 (4/1 대응)

### 혼합안 핵심
- **즉시(3월)**: 사람들 공장으로 설비 이전 (Plan D처럼 빠르게)
- **4월**: JV 설립 + 설비/현금 출자 구조화
- **5월~**: TIPS/연구과제 신청, 투자자 지분 전환 완료
- **하반기**: 파일럿 가동, 사람들 기존사업과 시너지 구축
