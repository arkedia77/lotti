Plan D: 영업양수도 (Asset Deal)

사람들이 로티의 설비 + IP만 매입. 로티 법인은 부채 정리 후 청산. 가장 단순하고 빠른 구조.

⚠ 구조적 제약: 설비 매각 ≠ 투자자 exit 자금

설비 매각대금은 은행 담보 8억 상환에 직행 → 투자자 exit 현금은 별도 자금원 필요. 설비만 매각해서는 투자자 exit이 맞물리지 않음. 사람들이 설비 매입가를 은행 상환액 + 투자자 exit분까지 포함하여 지불하거나, 별도로 투자자 exit 자금을 집행해야 함.

단점 / 리스크

  • 투자자 exit 가격 협상 난항 (실투입 45억 대비)
  • 정부자금(TIPS 등) 활용 불가
  • 이월결손금 73.3억 소멸 — 자산만 인수 시 법인세 절감 효과(~14.7억) 활용 불가. 로티 청산하면 결손금 소멸.

  • 사람들 과거 이력이 코스닥 상장심사에 리스크 → 사실상 상장 경로 차단 가능성
  • 200명 회원의 사람들 지분 참여 시 기존 캡테이블 복잡

영업양수도 세무 법인세법 §46(사업양수도), 부가세법 §10 검토 필요 포괄적 사업양수도 시 부가세 면제 가능. 개별자산 매각 시 부가세+양도차익 과세. 설비 장부가 vs 실거래가 차이 → 부당행위 부인 리스크.

Plan E: JV 신설 (합작법인)

사람들 + 로티 투자자들이 새 법인을 설립. 정부자금 극대화. 200명 회원은 펀드 또는 구주로 참여.

⚠ 자금 흐름 주의: 설비 매각대금 → 은행 직행

로티→JV 설비 매각대금은 은행 담보 8억 상환이 최우선. 투자자 exit 현금은 이 대금에서 나오지 않음 → 사람들/JV가 별도 자금으로 투자자 exit 집행 필요. exit과 설비 양수를 패키지로 동시 진행해야 구조가 성립.

Plan F: 혼합안 (추천) ★

JV 설립(E) + 영업양수도 속도(D) + 200명 조합 참여. 정부자금 + 깨끗한 구조 + 빠른 실행.

⚠ 핵심 자금 구조: 설비 매각 ≠ exit 자금

설비 매각대금(~8억)은 은행 담보 상환으로 직행. 투자자 exit은 사람들/JV가 별도 현금으로 집행해야 함. 따라서 설비 양수+은행 상환+투자자 exit을 하나의 패키지 딜로 동시 진행 설계 필요.

왜 Plan F인가?

  • 속도: 설비 이전 즉시 시작 (D와 동일). JV 설립은 병렬.
  • 정부자금: TIPS + 연구과제 = 최대 13억
  • 겸임 해결: 임대표는 로티 대표 유지 + JV에 기술자문계약.
  • 200명: 투자조합으로 묶어 JV에 출자 (49인 제한 → 4~5개 조합 분할, GP 유대표).
  • 투자자 유연성: exit(현금) + 전환(JV 지분) 선택.
  • 코스닥: 4년 로드맵, JV 단독 or 사람들 합병 후 상장 — 두 경로 열림.

Phase 2 (4~6월) — JV 설립 + 자산 양수도

항목법률 근거상태대응 방안
JV 설립 등기 상법 §288~§292 (발기설립) 표준 소요 1~2주. 사업목적에 "태양전지 및 에너지소재 제조·R&D·판매" 반드시 포함. 본점: 전남 나주(신규 기관투자 유치 시).
설비 양수도 vs 현물출자 상법 §299의2 (현물출자 검사), 법인세법 §46 양수도 추천 양수도(매매)가 현물출자보다 유리: 감정평가 불필요, 절차 간소. 현물출자 시 감정비 1~3천만원 + 3~4주 소요. 다만 투자자 지분 전환에는 현물출자 or 채권 전환 별도 구조 설계 필요.
RCPS → JV 지분 전환 구조 상법 §340(주식양도), 투자계약서 구조 설계 필요 RCPS가 로티 주식이므로 직접 JV 지분으로 전환 불가. 경로: ①로티가 설비 매각대금으로 RCPS 상환 → 상환금으로 JV 유증 참여, ②로티-JV 간 지분 스왑 계약. 세무/법무 자문 필수.
투자자 exit 가격 — 부당행위 부인 법인세법 §52, 상증세법 §35 주의 시가보다 현저히 낮은 가격(30%↑ 차이) 매각 시 부당행위 부인. 독립 감정평가서 확보 권장. 특수관계인 거래 해당 여부 검토.
투자조합 결성 (200명) 벤처투자법, 자본시장법 §9(사모), 코스닥 상장심사 지도 핵심 이슈 49인 초과 = 공모 해당 → 금융위 등록 필요.
