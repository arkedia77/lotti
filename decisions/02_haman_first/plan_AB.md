## 공통: 기존 기관 Exit 조건

| 투자자 | 실투입 | 40% 목표 | 현금 (50%) | 지분 유지 (50%) |
|--------|--------|----------|------------|-----------------|
| 에이지인베스트먼트 | 5억 | 2억 | 1억 | 1억 상당 지분 |
| 인라이트벤처스 | 20억 | 8억 | 4억 | 4억 상당 지분 |

- 현금 9억은 로티 부채 정리 후 잔여 자산 매각 or 신규법인/JV 자금으로 지급

## Plan A: 사람들 중심 신규법인 (함안) + 로티 인수

> **사람들이 중심이 되어 함안에 신규법인 설립 → 로티 사업 계승**
> **신규법인이 로티를 순차 인수 → 결손금 73.3억 즉시 활용 가능**
> **나주 이사 없음 / 전라도 펀드 불가 / 이후 기관펀딩 유효**

### 구조도

```
[Step 1] 설비 → 함안공장 이전 (공통)

[Step 2] 사람들 중심 신규법인 설립 (함안 소재)
         └── 유대표 + 사람들 출자

[Step 3] 기존 기관 exit 처리
         ├── 현금 9억 지급 (로티 자산 매각 대금 or 사람들 자금)
         └── 지분 9억 → 신규법인 주식으로 교환
              (기관이 신규법인의 소수주주로 잔류)

[Step 4] 신규법인 → 로티 법인 인수 (100% 지분 취득)
         └── 기관 구주 매수 완료 후 잔여 주주 정리

[Step 5] 신규법인 ← 로티 합병 (흡수합병)
         └── 결손금 73.3억 즉시 승계
             신규법인 매출에서 법인세 절감 시작

[Step 6] TIPS/연구과제 신청 (신규법인 자격)
         파일럿 가동 → 기관 펀딩 (Series A~)
```

### 기관 Exit 경로 (A안)

```
기존 기관 (RCPS 보유)
    │
    ├── [현금 50%] 로티 구주를 신규법인에 매각 → 현금 9억 수령
    │
    └── [지분 50%] 로티 구주 → 신규법인 신주로 교환 → 9억 상당 지분 보유
                    │
                    └── 신규법인 성장 시 추가 업사이드 향유
                        (코스닥 상장 시 exit 기회)
```

- **장점**: 기관이 신규법인에 직접 주주로 잔류 → 구조 단순
- **타이밍**: 로티 인수 시점(4~5월)에 일괄 처리 가능

### 결손금 73.3억 활용 (A안)

```
신규법인 (사람들 중심, 함안)
    │
    ├── 로티 100% 인수
    │
    └── 로티를 흡수합병
         └── 결손금 73.3억 승계 (법인세법 제44조의3)
             │
             ├── 요건: 사업 계속성 + 고용 승계 + 1년 이상 유지
             │   → 로티 설비+인력을 이미 운영 중이므로 충족 가능
             │
             └── 효과: 신규법인 매출(사람들 시너지+태양광)에서
                  향후 발생 이익과 상계 → 최대 14.7억 절감
```

- **시점**: 합병 완료 다음 사업연도부터 적용
- **A안 특이점**: 신규법인이 사람들과 함안에서 함께 운영 → 매출 발생 빠름 → 결손금 활용도 빠름

### 실행 단계

| 단계 | 내용 | 시기 |
|------|------|------|
| 1 | 함안공장 설비 이전 (공통) | ~4/1 |
| 2 | 신규법인 설립 (함안, 사람들 중심) | 4월 초 |
| 3 | RCPS 투자자 동의 + exit 조건 확정 | 4월 |
| 4 | 현금 9억 지급 + 지분 9억 교환 (기관 exit) | 4~5월 |
| 5 | 은행 담보대출 8억 상환 (설비 담보 해제) | 4~5월 |
| 6 | 신규법인 → 로티 100% 인수 완료 | 5월 |
| 7 | 로티 흡수합병 (결손금 승계) | 5~6월 |
| 8 | 임대표 CTO 합류, 파일럿 가동 | 4월~ |
| 9 | TIPS/연구과제 신청 | 5~6월 |
| 10 | 기관 펀딩 (Series A 등) | 하반기~ |

### 자금 흐름

| 항목 | 금액 | 비고 |
|------|------|------|
| 설비 이전비 | ~1억 | 함안 (1회) |
| 은행대출 상환 | 8억 | 설비 담보 해제 |
| 미지급금 처리 | 4억 | 퇴직금 등 |
| 기관 exit 현금 | 9억 | 실투입 40%의 50% |
| 신규법인 운영 | 10~15억 | 12개월 |
| **소계** | **32~37억** | |
| TIPS | -8억 (2년) | 신규법인 자격 |

| 결손금 절감 (장기) | -14.7억 | 합병 후 세제 수혜 |

### 장점
- **결손금 즉시 활용**: 로티 인수→합병이 빠르므로 절감 효과 조기 실현
- **나주 이사 불필요** → 설비 1회 이전, 공장 이중운영 없음
- **함안 시너지** → 사람들 알루미늄 + 태양광 수직계열화
- **기관 exit 단순**: 구주매수 + 신주교환 일괄 처리
- **TIPS/연구과제 가능** (신규법인 자격)

### 단점/리스크
- **전라도 펀드 30억 벨류 2억 불가** (나주 소재 요건 미충족)
- 기관 exit 현금 9억을 조기에 확보해야 함 (사람들 자금 부담)
- 합병 시 결손금 승계 요건 법률 검토 필수
- 사람들 관여도 높음 → 리딩방 이력 상장심사 노출 가능

## Plan B: 나주 JV 설립 (Plan J 계승)

> **함안 임시 보관 후 나주에 JV 신설 → Plan J 구조 계승**
> **전라도 펀드 + TIPS + 연구과제 활용 가능**
> **결손금은 JV→로티 합병으로 장기 승계**

### 구조도

```
[Step 1] 설비 → 함안공장 이전 (공통, 임시)

[Step 2] 나주에 JV 법인 설립
         유대표 800만 + 오브랩 200만 = Seed 1,000만원

[Step 3] AC 라운드: 1억 @ Post 10억

[Step 4] 사람들+VC 유증: Pre 20억
         사람들 15억 + VC 2억 (전라도 펀드, 30억 벨류)

[Step 5] 기존 기관 exit 처리
         ├── 현금 9억 (유증 자금에서 지급)
         └── 지분 9억 → JV 주식으로 교환

[Step 6] 은행 8억 상환 → 설비 담보 해제 → 설비 나주 이전

[Step 7] JV가 로티 설비+IP 인수 (영업양수도)

[Step 8] TIPS 신청 (8억, 2년)

[장기] JV → 로티 흡수합병 (결손금 승계) → 코스닥
```

### 기관 Exit 경로 (B안)

```
기존 기관 (RCPS 보유)
    │
    ├── [현금 50%] 로티 구주를 JV에 매각 → 현금 9억 수령
    │                                      (유증 자금에서 충당)
    │
    └── [지분 50%] 로티 구주 → JV 신주로 교환 → 9억 상당 지분 보유
                    │
                    └── JV 성장 시 추가 업사이드 향유
                        (코스닥 상장 시 exit 기회)
```

- **장점**: 유증 자금(17억)에서 현금 9억 충당 가능 → 별도 자금 불필요
- **타이밍**: 유증 완료 후(4~5월) 처리 → A안보다 약간 늦음
- **차이점**: 기관이 JV 주주로 잔류 (A안은 신규법인 주주)

### 결손금 73.3억 활용 (B안)

```
JV (나주)                     로티 (휴면)
    │                              │
    │   [2027~2028]                │
    └──────── 흡수합병 ────────────┘
                  │
                  └── 결손금 73.3억 승계
                      │
                      ├── 요건: 합병 전 1년+ 로티 사업 유지 필요
                      │   → 로티를 완전 휴면시키면 요건 미충족 리스크
                      │   → 최소한의 사업활동(설비 임대 등) 유지 필요
                      │
                      └── 효과: JV 매출 이익과 상계
                           → 최대 14.7억 절감 (A안과 동일)
```

- **시점**: 2027~2028 합병 후 → A안 대비 1~2년 늦음
- **B안 특이점**: 로티를 완전 휴면시키면 결손금 승계 요건 불충족 위험
  → 로티에 최소 사업활동 유지 비용 발생 (or 요건 포기)
