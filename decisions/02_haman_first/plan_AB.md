## Plan A: 사람들이 로티 합병 + 신규법인 설립 (함안)

> **사람들이 로티를 합병(주식교환) → 신규법인 설립 → 로티 사업 계승**
> **결손금 73.3억 수혜 + 함안공장 유지 (나주 이사 없음)**
> **단, 전라도 펀드(30억벨류 2억) 불가 / 이후 기관펀딩 유효**

### 구조도

```
[Step 1] 설비 → 함안공장 이전 (4/1까지)
         사람들이 이전비 ~1억 부담, 무상임대

[Step 2] 사람들이 로티를 합병 (주식교환 방식) — 4월 중
         ├── 기관 10% 현금 상환 (4억) ← 설비 매매 대금
         └── 기관 잔액 36억 → 신규법인 지분으로 교환

[Step 3] 미지급금 정리 (~3.16억 임금/퇴직금)
         임대표 미지급+가수금 (~5.37억) 처리

[Step 4] 사람들 중심 신규법인 설립 (함안) — 7월
         ├── 사람들 49% (창업기업 요건: 모회사 49% 이하)
         └── 로티 설비+IP → 사람들이 매도 → 신규법인 인수

[Step 5] 벤처투자 1차: 2~5억
[Step 6] 벤처투자 2차: 10억
[Step 7] TIPS 신청 (9월~)

[장기] 결손금 73.3억 → 합병으로 승계 → 세제 수혜
```

### 기관 Exit 경로 (A안)

```
기존 기관 (RCPS 보유)
    │
    ├── [10%] 현금 상환 → 4억 수령
    │         (설비 매매 대금에서 충당)
    │
    └── [90%] 로티 구주 → 신규법인 주식으로 교환
              │         (36억 상당 지분)
              │
              └── 신규법인 성장 시 업사이드
                  코스닥 상장 시 exit
```

### 결손금 73.3억 활용 (A안)

```

    ├── 로티 결손금 73.3억 직접 승계
    │   → 사람들 매출(95.8억+)에서 법인세 절감

    │
    └── 장점: 사람들이 이미 매출 있으므로 즉시 활용 가능
```

**A안 결손금 강점**: 사람들이 로티를 합병하면 사람들의 기존 매출(95.8억)로 결손금을 **즉시** 상계 가능

## Plan B: 나주 JV 설립 (Plan J 계승)

> **함안 임시 보관 후 나주에 JV 신설 → Plan J 구조 계승**
> **전라도 펀드 + TIPS + 연구과제 활용 가능**
> **결손금은 JV→로티 합병으로 장기 승계**

### 구조도

```
[Step 1] 설비 → 함안공장 이전 (공통, 임시)

[Step 2] 나주에 JV 법인 설립
         유대표 800만 + 오브랩 200만 = Seed 1,000만원

[Step 3] AC 라운드: 1억 @ Post 10억

[Step 4] 사람들+VC 유증: Pre 20억
         사람들 15억 + VC 2억 (전라도 펀드, 30억 벨류)

[Step 5] 기관 exit
         ├── 10% 현금 상환 (4억) ← 유증 자금
         └── 잔액 36억 → JV 주식으로 교환

[Step 6] 은행 8억 상환 → 설비 담보 해제 → 설비 나주 이전

[Step 7] JV가 로티 설비+IP 인수 (영업양수도)

[Step 8] TIPS 신청 (8억, 2년)

[장기] JV → 로티 흡수합병 (결손금 승계) → 코스닥
```

### 기관 Exit 경로 (B안)

```
    │

              └── JV 성장 시 업사이드
```
